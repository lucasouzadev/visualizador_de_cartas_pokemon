## Este projeto traz a ideia de uma visualização 3D web de cartas pokemon.

# Funcionabilidades

- Banco de imagens pokemon
- Animações de apresentação
- Interação animada
- Opções para alterar animações e cartas

# Tecnologias

- HTML
- CSS
- JavaScript
